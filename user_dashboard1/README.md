# 🌟 Magnevents — Premium Live Artist Booking Platform

[![Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![React](https://img.shields.io/badge/React-18-blue?style=for-the-badge&logo=react)](https://reactjs.org/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-Latest-purple?style=for-the-badge&logo=framer)](https://www.framer.com/motion/)
[![Supabase](https://img.shields.io/badge/Supabase-Database-green?style=for-the-badge&logo=supabase)](https://supabase.com/)

Magnevents is a high-end, luxury-themed platform designed to bridge the gap between elite live performers and event organizers. Whether it's a grand wedding, a high-stakes corporate gala, or an intimate concert, Magnevents provides a seamless, aesthetically stunning experience for booking the perfect talent.

---

## ✨ Key Features

### 🏆 Premium Visual Excellence
- **Gold Luxury Theme**: A curated dark-mode aesthetic featuring deep obsidian backgrounds and vibrant gold accents.
- **Glassmorphism**: Modern UI elements with soft blurring and translucent overlays for a high-end feel.
- **Micro-Animations**: Powered by **Framer Motion**, ensuring every interaction feels fluid and responsive.

### 📅 Advanced Booking Flow
- **Intuitive Navigation**: A streamlined "How to Book" process that guides users from discovery to confirmation.
- **Dual Modal System**:
  - **Gold Luxury Modal**: For client inquiries and artist bookings.
  - **Sky Blue Portal**: A dedicated registration flow for artists to join the elite roster.

### 📱 Mobile-First Architecture
- **Responsive Navigation**: A custom-built bottom navigation bar for mobile users, mirroring the premium desktop experience.
- **Adaptive Layouts**: Seamless transitions between desktop, tablet, and mobile views without compromising on style.

### 🚀 Performance & SEO
- **Next.js 14 App Router**: Utilizing the latest React Server Components for lightning-fast initial page loads.
- **Dynamic Optimization**: Intelligent code-splitting and dynamic imports for below-the-fold content.
- **SEO Optimized**: Semantic HTML and optimized metadata for maximum search engine visibility.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Library**: [React 18](https://reactjs.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Vanilla CSS with CSS Variables (Tokens)
- **State Management**: [TanStack Query](https://tanstack.com/query/latest)
- **Backend/DB**: [Supabase](https://supabase.com/)
- **API Client**: Axios

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AvneeshTripathi72/EventFlow.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hello121-main
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
